This is a project for my sister who writes all the time. It is supposed to be a personal website for her to display her work effectively and to be copied and developed later by me for a more full website idea. It will have a mostly simple function and some complex ones. The main idea however is to make it automatically be updated. She can write in Google Docs or whatever and it automatically add an entry and display it on the front page. Then it will archive the old ones into google probably so they can be accessed later.
